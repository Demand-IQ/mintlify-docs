This guide walks you through creating a deck, adding slides, and generating a shareable presentation link using the Sales CoPilot API.

## Prerequisites

- A valid session cookie from `POST /api/auth/login`
- Your organization must be provisioned in Sales CoPilot

## Step 1: Create a deck

Call `POST /api/decks` with a name and deck type. Use `image-based` if you plan to upload slide images for AI processing, or `structured` for manual slide creation.

```typescript
const res = await fetch("https://app.demand-iq.com/api/decks", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    name: "Solar Roofing Pitch",
    type: "image-based",
    deckContext: "Residential solar installation for homeowners in the Southwest US"
  }),
});

const { deck } = await res.json();
console.log(deck.id); // e.g. "a1b2c3d4-..."
```

## Step 2: Upload slides and generate content

For image-based decks, upload your slide images to `POST /api/decks/{deckId}/generate`. Sales CoPilot analyzes each image, extracts content, writes narration scripts, and synthesizes audio.

```typescript
const formData = new FormData();
formData.append("images", slideFile1);
formData.append("images", slideFile2);

const res = await fetch(
  `https://app.demand-iq.com/api/decks/${deck.id}/generate`,
  { method: "POST", body: formData }
);

const { jobId } = await res.json();
```

Poll `GET /api/decks/{deckId}/generate/status` until the job completes:

```typescript
const status = await fetch(
  `https://app.demand-iq.com/api/decks/${deck.id}/generate/status`
).then(r => r.json());

console.log(status.status); // "pending" | "processing" | "completed" | "failed"
```

## Step 3: Configure branding

Customize the presentation appearance with `PUT /api/decks/{deckId}/branding`:

```typescript
await fetch(`https://app.demand-iq.com/api/decks/${deck.id}/branding`, {
  method: "PUT",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    palette: {
      primary: "#1a73e8",
      primaryLight: "#4a9af5",
      accent: "#fbbc04",
      neutral: "#5f6368",
      text: "#202124"
    },
    cta: {
      headline: "Ready to go solar?",
      body: "Schedule your free consultation today.",
      buttonText: "Book now",
      buttonUrl: "https://example.com/book"
    }
  }),
});
```

## Step 4: Create a presentation

Generate a personalized, shareable presentation link with `POST /api/decks/{deckId}/presentations`:

```typescript
const res = await fetch(
  `https://app.demand-iq.com/api/decks/${deck.id}/presentations`,
  {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      homeowner: {
        first_name: "Jane",
        last_name: "Smith",
        address: "123 Main St",
        city: "Phoenix",
        state: "AZ",
        zip: "85001",
        country: "US",
        email: "jane@example.com"
      }
    }),
  }
);

const { url, status } = await res.json();
console.log(url); // Shareable presentation URL
```
